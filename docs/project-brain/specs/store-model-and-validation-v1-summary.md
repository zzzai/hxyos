## 当前口径

这不是已审计财务结果，而是样板店假设模型。

计算公式：
- 套餐月营收 = `套餐价格 * 月订单数`
- 月总营收 = `套餐月营收合计 + 其他月收入`
- 月变量成本 = `套餐月营收合计 * 变量成本率`
- 月净现金流 = `月总营收 - 月变量成本 - 月固定成本`
- 回本周期 = `初始投资 / 月净现金流`，向上取整

## 样例结果

基于当前样例输入：
- 初始投资：180000
- 月固定成本：68000
- 变量成本率：28%
- 月总营收：142760
- 月净现金流：36467.2
- 回本周期：5 个月

## 套餐结构

样例套餐：
- 基础款：68 元，月 320 单
- 招牌款：128 元，月 620 单
- 尊享款：198 元，月 180 单

当前样例里，招牌款贡献最高，月营收 79360，占月总营收约 55.59%。

## 样板店验证矩阵

当前纳入 7 个验证项：
- 品牌资产一致率
- 无提示定位复述率
- 套餐选择率
- 月净现金流
- 回本周期
- 复购率
- 健康档案完整率

## 边界

- 真实回本周期必须用真实房租、人工、耗材、客流、套餐结构和现金到账数据复核。
- 医疗诊断、长期健康结果和平台化收入不进入当前小店模型。
- 这是样板店阶段工具，不能替代财务尽调和正式预算。
