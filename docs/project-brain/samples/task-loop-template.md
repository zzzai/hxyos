# HXY Task Loop Template

## Goal

One measurable sentence:

- what should improve
- by how much
- by when

## Context

Only the active facts:

- current asset
- current constraint
- current owner
- current evidence gap

## Tool

What will actually run:

- test
- model
- search
- ingest
- review
- update

## Evaluation

How success is judged:

- pass / fail
- score
- reviewer sign-off
- metric delta

## Stop Condition

Choose one:

- success threshold reached
- max iterations reached
- evidence insufficient
- target drift detected

## Next Output

Return only:

- conclusion
- evidence gap
- next action
- owner
- version update
