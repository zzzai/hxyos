## 事实与假设边界

| 类别 | 当前结论 | 状态 | 验证方法 |
|---|---|---|---|
| 当前主定位 | 社区泡脚按摩小店 | 可作为 v1 经营口径 | 顾客无提示复述率、首访转化 |
| 融资叙事 | 社区健康服务入口 | 可作为对投资人表达 | 样板店复购、健康档案、私域留存 |
| 远期愿景 | 银发健康科技平台 / 社区健康数字基础设施 | 只作为长期愿景 | 多店模型、数据闭环、组织复制能力 |
| 核心口号 | 草本真现煮，按出真功夫 | 待终端测试 | 门头识别、顾客复述、团购点击 |
| 单店回本 | 5-8 个月区间假设 | 待真实参数替换 | 初始投资、房租、人工、现金流 |
| 套餐结构 | 基础款 / 招牌款 / 尊享款 | 待样板店验证 | 套餐选择率、客单价、复购率 |
| 私域复购 | 服务后护理建议带动复购 | 待验证 | 7/30 天跟进转化、二访率 |

## 问答使用规则

1. 回答 HXY 项目问题时，优先引用 `projects/hxy/deliverables/` 和 `knowledge/hxy/structured/`。
2. 涉及品牌理论时，再合并 `knowledge/brand/index.json`。
3. 涉及营销动作时，再合并 `knowledge/marketing/index.json`。
4. 不把融资愿景当成当前门店承诺。
5. 不把未经验证的单店财务假设当成已发生数据。
6. 输出建议必须落到门店动作、验证指标和风险边界。
