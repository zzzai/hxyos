# HXY 超智大脑阶段路线

## 阶段 0：项目知识资产化

目标：让系统真正懂 HXY。

交付：

- 知识资产清单
- claim 抽取
- 实体关系
- 版本治理
- 冲突识别
- HXY + 华与华联合问答

## 阶段 1：筹备期大脑

目标：把第一家店为什么能开、怎么赚钱讲清楚。

交付：

- 品牌策划全案
- 小店模型计算器
- 产品结构
- 样板店开店手册
- 财务假设清单
- 选址评分模型 v1

## 阶段 2：样板店大脑

目标：验证真实模型。

交付：

- 样板店日报
- 假设验证看板
- 产品复购分析
- 价格带验证
- 用户反馈分析
- 店长动作闭环

## 阶段 3：10-50 店复制大脑

目标：把成功模型标准化。

交付：

- 门店评级
- 店长画像
- 加盟商评分
- SOP 执行率
- 督导任务
- 开业爬坡模型

## 阶段 4：100-1000 店区域大脑

目标：区域规模化。

交付：

- 城市分层
- 区域经营模型
- 供应链预测
- 加盟商风控
- 区域品牌资产
- 区域经理 Agent

## 阶段 5：万店规模大脑

目标：全国连锁智能操作系统。

交付：

- 自动选址
- 自动加盟评估
- 自动经营异常发现
- 自动巡店
- 自动 SOP 更新
- 自动区域策略生成
- 自动品牌资产监控
